# ATS-Insight-Smart-Resume-Analyzer
ATS-Insight is a JavaFX-based Smart Resume Analyzer that simulates the working of an Applicant Tracking System (ATS). The application analyzes resumes, detects technical skills, evaluates ATS compatibility, and generates intelligent suggestions to improve resume quality for placements and job applications.

# Features
- Resume Analysis and ATS Scoring
- Technical Skill Detection
- Smart Resume Improvement Suggestions
- Modern JavaFX User Interface
- Fast and Lightweight Desktop Application
- User-Friendly Design

# Technologies Used
- Java
- JavaFX
- CSS
- IntelliJ IDEA

# Project Objective
The main objective of this project is to help students and job seekers improve their resumes according to ATS standards used by companies during recruitment.

# How It Works
1. Upload Resume
2. Resume Text Extraction
3. Skill and Keyword Analysis
4. ATS Score Generation
5. Smart Suggestions Display
